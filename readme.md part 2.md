# Baker's Delights

Freshly baked goodness — a simple, responsive bakery landing page showcasing menu items, contact info, and the bakery story.

## Overview

This repo contains a static HTML/CSS site for Baker's Delights. It's a clean, responsive single-page layout built with semantic HTML and modern CSS (custom properties, grid, and flex). The site includes a hero, about section, featured menu, and contact details.

## Tech Stack

- HTML5
- CSS3 (custom properties, grid, flexbox)
- Google Fonts: Playfair Display & Poppins

No build tools are required — this is a plain static site.
